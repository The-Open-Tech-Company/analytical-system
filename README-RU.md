# Аналитическая система

Комплексная система для анализа, сравнения и распознавания лиц и голосов.

**Репозиторий:** https://github.com/The-Open-Tech-Company/analytical-system  
**Лицензия:** Unlicense (Открытый исходный код)

## Возможности

### Анализ лиц
- **Извлечение характеристик лица**: Извлечение биометрических характеристик с использованием MediaPipe Face Mesh
- **Определение пола, возраста, расы**: Продвинутое определение с использованием нескольких методов (DeepFace, DNN, эвристика)
- **Сравнение лиц**: Детальное сравнение двух лиц с процентами совпадения
- **Визуализация**: Интерактивная визуализация характеристик лица и результатов сравнения
- **База данных лиц**: Хранение и поиск лиц в базе данных SQLite

### Анализ голоса
- **Извлечение характеристик голоса**: Извлечение акустических характеристик (высота тона, MFCC, форманты и др.)
- **Определение пола**: Определение пола говорящего по характеристикам голоса
- **Определение языка**: Определение языка речи с использованием акустического анализа и распознавания речи
- **Определение акцента**: Определение акцента
- **Определение эмоций**: Определение эмоционального состояния по голосу
- **Сравнение голосов**: Сравнение двух голосовых образцов на схожесть

## Требования

- Python 3.11+
- OpenCV (opencv-contrib-python)
- MediaPipe
- NumPy
- TensorFlow (для DeepFace)
- Librosa (для анализа голоса)
- SpeechRecognition (опционально, для определения языка)

## Установка

```bash
pip install -r requirements.txt
```

## Использование

### Графический интерфейс (Рекомендуется)

```bash
python gui_app.py
```

### Командная строка

```bash
python main.py <путь_к_изображению1> <путь_к_изображению2>
```

## Структура проекта

- `gui_app.py` - Главное GUI приложение
- `main.py` - Интерфейс командной строки
- `face_analyzer.py` - Извлечение характеристик лица
- `face_comparator.py` - Логика сравнения лиц
- `face_visualizer.py` - Визуализация лиц
- `voice_analyzer.py` - Извлечение и анализ характеристик голоса
- `voice_comparator.py` - Логика сравнения голосов
- `face_database.py` - Управление базой данных лиц
