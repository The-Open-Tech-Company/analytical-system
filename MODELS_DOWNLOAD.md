# Инструкция по загрузке моделей для OpenCV DNN

Если автоматическая загрузка моделей не работает (ошибка 404), выполните следующие шаги для ручной загрузки:

## Способ 1: Загрузка через GitHub (рекомендуется)

### Шаг 1: Загрузка файлов конфигурации (.prototxt)

1. Перейдите на страницу репозитория OpenCV:
   - https://github.com/opencv/opencv/tree/master/samples/dnn/face_detector

2. Скачайте следующие файлы:
   - `deploy_gender.prototxt` - конфигурация модели пола
   - `deploy_age.prototxt` - конфигурация модели возраста

   **Как скачать:**
   - Нажмите на имя файла
   - Нажмите кнопку "Raw" (или "Сырой")
   - Сохраните файл (Ctrl+S или правой кнопкой мыши → Сохранить как)

### Шаг 2: Загрузка файлов моделей (.caffemodel)

1. Перейдите на страницу репозитория OpenCV Extra:
   - https://github.com/opencv/opencv_extra/tree/master/testdata/dnn

2. Скачайте следующие файлы:
   - `gender_net.caffemodel` - модель для определения пола (~3-5 МБ)
   - `age_net.caffemodel` - модель для определения возраста (~3-5 МБ)

   **Примечание:** Эти файлы могут быть большими. Если GitHub показывает ошибку, используйте Git LFS или Способ 2.

### Шаг 3: Размещение файлов

Поместите все 4 файла в папку `models/` в корне проекта:

```
systa/
  models/
    deploy_gender.prototxt
    gender_net.caffemodel
    deploy_age.prototxt
    age_net.caffemodel
```

## Способ 2: Использование Git LFS

Если файлы .caffemodel не загружаются через браузер, используйте Git LFS:

```bash
# Установите Git LFS (если еще не установлен)
git lfs install

# Клонируйте репозиторий OpenCV Extra
git clone https://github.com/opencv/opencv_extra.git
cd opencv_extra/testdata/dnn

# Скопируйте файлы в папку models/
cp gender_net.caffemodel age_net.caffemodel ../../../../models/
```

## Способ 3: Альтернативные источники

Если GitHub недоступен, попробуйте найти модели на:
- Официальном сайте OpenCV: https://opencv.org/
- Зеркалах репозиториев OpenCV
- Форумах и сообществах разработчиков

## Проверка установки

После размещения файлов в папке `models/`, перезапустите приложение. Модели должны загрузиться автоматически.

Если проблемы сохраняются, убедитесь, что:
1. Все 4 файла находятся в папке `models/`
2. Имена файлов написаны правильно (без опечаток)
3. Файлы не повреждены (проверьте размер файлов .caffemodel - они должны быть ~3-5 МБ)

## Примечание

Модели .caffemodel могут быть слишком большими для прямого скачивания через GitHub web-интерфейс. В этом случае используйте Git LFS или альтернативные источники.
